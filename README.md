# FUTURE_DS_01

Project Title:

Superstore Revenue Analysis Dashboard


Project Overview :

An interactive Power BI dashboard analyzing sales trends ,product performance , and regional distribution for a retail superstore.


Key Insights & Recommendations :
 

1. Revenue Trends :

Observation : Significant decline after 2014 ,followed by a stagnant plateau through 2017.

Recommendation : Investigate the loss of high-value 2014 accounts and launch a re-acquisition campaign.


2.Product Performence :

Observation : The canon imaging line is the primary revenue driver ,nearly doubling the sales of any other product.

Recommendation : Use the Canon brand as a "lead magnet" to bundle slower-moving accessories.


3.Regional Analysis :

Observation : West and East regions contribute - 70 % of total revenue.

Recommendation :Focus marketing spend on high-performing West/East regions to ROI.
